# GJames.V6.ExtensionEngine.Demo

Demo solution for a modular V6 extension engine.

This repo is used to prove the workflow:

* Multiple .NET Framework 4.5.1 DLL projects
* GitHub builds the DLLs (not local dev machines)
* Releases are shipped as versioned bundles (ZIP)
* Bundles are installed into the Citrix master image
